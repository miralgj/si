# Si

Si exposes your commands and scripts as a RESTful API.

Is this a bad idea? Si!

## Command-Line Options

* `--command` 
  Command to expose (can be used multiple times)

* `--list-host` 
  Specifies the host to listen on 
  Default: `0.0.0.0`

* `--port` 
  Port to listen on 
  Default: `3000`
